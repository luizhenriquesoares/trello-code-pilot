# Changelog

## 0.2.0

- Real-time Output panel — watch agent execution live
- Parallel execution — run 2 or 3 agents simultaneously
- Open card in Trello directly from VS Code
- Rich card tooltips with Markdown (labels, due date, checklists)
- Checklist progress indicator in sidebar (e.g., 3/5)
- Overdue warning icons on cards
- Branch fallback — switches to existing branch if name conflicts
- Execution duration tracking
- `.vscodeignore` for smaller extension package

## 0.1.0

- Initial release
- Trello board ↔ repo mapping via setup wizard
- Sidebar with cards organized by list
- Run Claude Code agent on single card or all cards
- Auto branch creation and card movement
- Secure credential storage
